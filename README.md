## `실행방법`
`$ git clone https://github.com/mye0ngs00/redux_counter_2020`<br/>
`$ cd redux_counter_2020`<br/>
`$ npm install && npm start`<br/>


### `스택`

- react-redux
- redux-actions
- react

### `기능`
- 좌클릭 - 숫자 증가
- 우클릭 - 숫자 감소
- 더블클릭 - 색상 변환
- 원 추가, 제거
